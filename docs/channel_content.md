### #はじめにお読みください

**チャンネル説明欄（トピック）**: サーバールール・参加ガイド・ポイント制度のご案内

**投稿内容（メッセージ①：ようこそ）**:

```

💡 **このサーバーでできること**
• 製品の使い方やトラブルについて質問・回答
• 他のユーザーの質問に回答してポイントを獲得
• ポイントを貯めてノベルティや製品割引クーポンをゲット

📌 **まずこのチャンネルの内容をすべてお読みください。**
ルールを確認したら、下の ✅ リアクションを押して参加しましょう！
```

**投稿内容（メッセージ②：基本ルール）**:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

このサーバーでは、すべてのメンバーが快適に参加できるよう以下のルールを設けています。

**1️⃣ 相互尊重**
すべてのメンバーに敬意を持って接してください。個人攻撃・侮辱・差別的発言は禁止です。

**2️⃣ 正確な情報を**
技術的な回答は、確認済みの情報を投稿してください。不確かな場合は「〜だと思います」と明記をお願いします。

**3️⃣ チャンネルの正しい使い方**
各チャンネルの用途に沿った投稿をしてください。質問は製品カテゴリに合ったサポートチャンネルへ。

**4️⃣ 解決したら /solved**
質問が解決したら、必ず `/solved` コマンドを実行してください。回答者にポイントが付与されます。

**5️⃣ 禁止事項**
🚫 スパム・荒らし行為
🚫 個人情報の無断掲載
🚫 違法コンテンツの共有
🚫 競合他社製品の中傷
🚫 無関係な宣伝・広告
```

**投稿内容（メッセージ③：ポイント制度）**:

### #discord-使い方ガイド

**チャンネル説明欄（トピック）**: Discord 初心者向けの操作ガイド

**投稿内容（メッセージ①：基本操作）**:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📖 Discord の使い方ガイド
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Discord が初めての方向けに、基本操作をまとめました。
わからないことがあれば #雑談 でお気軽に聞いてください！

📌 **サーバーの構造**
• **カテゴリ**（📢 お知らせ、❓ テクニカルサポート 等）= チャンネルのグループ
• **チャンネル**（# マーク）= 話題ごとの部屋
• **スレッド** = チャンネルの中の個別トピック（質問ごとの小部屋）

📌 **メッセージの送り方**
• 画面下の入力欄にテキストを入力して Enter で送信
• Shift + Enter で改行（送信せずに改行）
• 画像はドラッグ＆ドロップ or ＋ ボタンから「ファイルをアップロード」で添付
```

**投稿内容（メッセージ②：質問の仕方）**:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
❓ テクニカルサポートでの質問方法
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

サポートチャンネル（#マザーボード-サポート 等）は**フォーラム形式**です。
掲示板のように、質問ごとにスレッドを作って投稿します。

📝 **質問の手順**
1. 左のチャンネル一覧から、質問に合うサポートチャンネルをクリック
2. 画面上部の「**投稿を作成**」ボタンをクリック
3. **タイトル**に質問の概要を入力（例:「BIOSが起動しない」）
4. **本文**に詳細を記入（製品名・症状・試したこと）
5. 「投稿」ボタンで送信！

📝 **解決したら**
質問が解決したら、スレッド内で `/solved` と入力して送信してください。
回答してくれた方にポイントが付与されます ✅

> 📱 スマホの場合は右下の ＋ ボタンからスレッドを作成できます。
```

**投稿内容（メッセージ③：便利な機能）**:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💡 知っておくと便利な機能
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔹 **リアクション（👍 等）**
メッセージにカーソルを合わせると顔文字アイコンが出ます。
クリックしてリアクションを選べます。
役に立った回答に 👍 を付けると、回答者にポイント +1 pt！

🔹 **メンション（@ユーザー名）**
特定の人に話しかけたいときは @ を付けて名前を入力。
相手に通知が届きます。

🔹 **スラッシュコマンド**
入力欄に `/` を打つと、使えるコマンド一覧が表示されます。
• `/mypoints` → 自分のポイントを確認
• `/leaderboard` → ランキングを見る

🔹 **ピン留めメッセージ**
チャンネル上部の 📌 アイコンから、重要なメッセージを確認できます。

📖 Discord 公式ヘルプ: https://support.discord.com/hc/ja
```
