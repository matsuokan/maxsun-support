# MAXSUN コミュニティサポート プロジェクト概要

> **対象製品**: マザーボード / グラフィックボード（MAXSUNブランド）  
> **方式**: Discord を基盤とした既存ユーザー主導のピアサポートコミュニティ  
> **目標**: サポートコストの削減と、貢献ユーザーへの売上還元による持続可能なコミュニティ育成

## ✅ 決定事項（2026-03-19）

| 項目 | 決定内容 |
|---|---|
| システム | Discord サーバー + カスタム Bot。データ管理は **Google Sheets** |
| 初期サポーター | **MAXSUN 社員**が担当（コールドスタート対策） |
| 還元方法 | **ノベルティグッズ** と **製品割引クーポン** のみ（現金還元なし） |

---

## 1. コンセプト

```
新規ユーザー → 質問 → [Discord コミュニティ] → 既存ユーザーが回答
                                                       ↓
                                            貢献ポイント加算
                                                       ↓
                                     MAXSUN 売上の一部を還元（報酬）
```

- **新規ユーザー**: 製品の使い方やトラブルを質問する
- **サポーター（既存ユーザー）**: 回答・解決支援を行い、貢献度を積み上げる
- **MAXSUN 運営**: コミュニティを監督しつつ、貢献者に報酬を還元する

## 3. 貢献度スコアシステム

### ポイント加算ルール（例）

| アクション | 加算ポイント |
|---|---|
| 質問スレッドで回答を投稿 | +5 pt |
| 質問者が「解決済み」マークを付与 | +20 pt |
| 他ユーザーからのリアクション（👍）each | +1 pt |
| FAQ・Wikiへの記事投稿が承認 | +50 pt |
| 不正・誤情報の提供（ペナルティ） | −10 pt |

### ランク制度

| ランク | 必要ポイント | 特典 |
|---|---|---|
| 🥉 ヘルパー | 0〜199 | コミュニティ参加権 |
| 🥈 サポーター | 200〜999 | 専用バッジ・先行情報 |
| 🥇 エキスパート | 1,000〜4,999 | 製品割引クーポン |
| 💎 アンバサダー | 5,000〜 | 売上還元報酬・製品提供 |

## 4. 還元スキーム（報酬設計）✅ 決定済み

### 採用する還元方法

| ランク | 還元内容 |
|---|---|
| 🥈 サポーター（200pt〜） | MAXSUN ロゴ入りノベルティ（ステッカー・アクスタ等） |
| 🥇 エキスパート（1,000pt〜） | 製品割引クーポン（10〜20%OFF） |
| 💎 アンバサダー（5,000pt〜） | 上位クーポン（30%OFF） + 限定ノベルティセット |

> [!NOTE]
> 現金・ギフトカード還元は**初期フェーズでは実施しない**。  
> コミュニティが成熟した段階で改めて検討する。

### 運用フロー
1. 毎月末に Google Sheets でポイント集計
2. ランク到達者をリストアップ → 運営が手動でクーポンコード / ノベルティ発送を処理

## 5. 必要なシステム構成 ✅ 決定済み

### 採用構成（MVP）

```
Discord サーバー
  └── Discord Bot（Python / discord.py）
        ├── /solved コマンド → 質問者が解決マークを付ける
        ├── /mypoints コマンド → 自分のポイント確認
        ├── ✅ リアクション検知 → ポイント自動加算
        └── Google Sheets API 連携（ポイントDB）
```

**Google Sheets 管理シート（想定）**

| シート名 | 内容 |
|---|---|
| `members` | Discord ID, 名前, 累計ポイント, ランク |
| `point_log` | 日時, ユーザー, アクション, 加算ポイント |
| `rewards_log` | 還元履歴（クーポンコード, 発送状況） |

### 将来的な拡張（未決定）
- 管理ダッシュボード（Web UI）
- FAQ 自動生成 / AI アシスト回答
- DB を PostgreSQL へ移行

## 6. 運営体制 ✅ 決定済み

**初期フェーズは全員 MAXSUN 社員で担当する。**

| 役割 | 担当 | 主な業務 |
|---|---|---|
| コミュニティマネージャー | 社員 1名 | 全体監督・ルール整備・報酬処理 |
| テクニカルサポーター | 社員（エンジニア） | 質問回答・FAQ 作成 |
| モデレーター | 社員 1名 兼務 | スパム対応・スレッド整理 |

> [!NOTE]
> コミュニティが成長し、信頼できる上位ランクユーザーが増えたタイミングで  
> モデレーター権限を外部コントリビューターへ段階的に移譲する。

## 7. 考慮が必要な事項

### ✅ 品質管理
- 誤情報の拡散防止のため、**テクニカルレビュアーによる回答検証ルート**を設ける
- 解決マークは質問者のみが付けられる仕組みにする（自作自演防止）

### ✅ 不正対策
- 自己回答によるポイント不正取得の防止（同一 IP / アカウント検知）
- スパム・荒らし行為へのペナルティと Ban 基準を規約に明記

### ✅ 初期コミュニティ形成（コールドスタート問題）
- 立ち上げ初期はユーザーが少ないため、MAXSUN 社員がサポーターを兼任
- 製品購入者へのメール・同梱物でのコミュニティ招待が有効
- ベータ期間中はポイント 2〜3 倍キャンペーンで参加を促す

### ✅ 多言語対応
- MAXSUN のグローバル展開を考慮し、日本語・英語・中国語チャンネルを段階的に整備

### ✅ コンプライアンス
- 報酬還元の法的位置づけ（謝礼か業務委託か）を明確化
- 個人情報（報酬振込先等）の取り扱いポリシーの策定
