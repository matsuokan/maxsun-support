## 2. チャンネル構成


---

### 🆕 はじめに

> 新規参加者が最初に確認するカテゴリ。ゲートウェイとして機能させる。

| チャンネル名 | 説明 | 権限 |
|---|---|---|
| `#はじめにお読みください` | サーバールール・参加ガイド・ポイント制度の説明 | 全員読み取り、運営のみ投稿 |

### ❓ テクニカルサポート

> サポートの中核。フォーラム形式（スレッド）で運用する。

## 3. ロール（役割）設計

### ロール一覧と権限

| ロール名 | 色 | 付与条件 | 主な権限 |
|---|---|---|---|
| `👑 MAXSUN 公式` | 金色 | MAXSUN 社員 | 全チャンネル管理・投稿 |
| `🛡️ モデレーター` | 青 | 運営が任命 | メッセージ削除・タイムアウト |
| `💎 アンバサダー` | 紫 | 5,000 pt 到達 | 専用バッジ表示・限定チャンネル閲覧 |
| `🥇 エキスパート` | 金 | 1,000 pt 到達 | 専用バッジ表示 |
| `🥈 サポーター` | 銀 | 200 pt 到達 | 専用バッジ表示 |
| `🥉 ヘルパー` | 緑 | 参加時に自動付与 | 基本チャンネル閲覧・投稿 |
| `🤖 Bot` | グレー | Bot アカウント | Bot 用チャンネルへの投稿 |

### ロール付与フロー

```
参加
 └─ 🥉 ヘルパー（自動付与）
       └─ 200 pt 達成 → 🥈 サポーター（Bot 自動昇格）
             └─ 1,000 pt 達成 → 🥇 エキスパート（Bot 自動昇格）
                   └─ 5,000 pt 達成 → 💎 アンバサダー（Bot 自動昇格）
```

## 4. Bot の役割（サーバー内での動作）

Bot の詳細仕様は [`bot_spec.md`](bot_spec.md) を参照。

| Bot の動作 | トリガー | 動作場所 |
|---|---|---|
| 質問テンプレート表示 | フォーラムでスレッド作成時 | `#*-サポート` 各チャンネル |
| `/solved` 実行 → ポイント加算 | 質問者がコマンド実行 | サポートスレッド内 |
| `👍` リアクション → ポイント加算 | 他ユーザーがリアクション | 全サポートチャンネル |
| `/mypoints` → ポイント表示 | ユーザーがコマンド実行 | 全チャンネル（Ephemeral） |
| 週次ランキング投稿 | 毎週日曜 0:00 JST | `#ポイントランキング` |
| ランク自動昇格通知 | ポイント閾値達成時 | `#ポイントランキング` |
| 報酬申請受付 | `/reward-apply` コマンド | `#受け取り申請` |
| モデレーションログ | スパム・違反検知時 | `#モデレーション-ログ`（非公開） |

## 5. 運用ルール（サーバー規約）

`#はじめにお読みください` に掲載するルールの骨子。

### 基本ルール

1. **敬意を持ったコミュニケーション**を心がけてください
2. 個人攻撃・差別的な発言は禁止です
3. スパム・宣伝行為は禁止です
4. 質問には **製品名・型番・症状** を明記してください
5. 解決したら必ず `/solved` コマンドで解決済みにしてください

### ポイントに関するルール

- 自分で質問して自分で解決済みにすることは**禁止**（自作自演防止）
- 意図的な誤情報の提供は**ペナルティ（−10 pt）**の対象
- 悪質な不正行為はアカウント停止・Ban 対象

### 禁止事項

- 競合他社製品の中傷
- 違法なソフトウェアや改造に関する情報の共有
- 個人情報の無断掲載

## 6. 参加フロー（ユーザー体験）

```
①  Discord の招待リンクからサーバーに参加
        ↓
②  #はじめにお読みください でルールを確認
        ↓
③  ✅ リアクションでルール同意 → 🥉 ヘルパー ロールが自動付与
        ↓
④  質問がある → 製品カテゴリの #*-サポート にスレッドを立てる
   回答できる → 他ユーザーのスレッドに回答する
        ↓
⑤  質問者が解決したら /solved を実行 → 回答者に +20 pt
        ↓
⑥  ポイントが貯まる → ランク自動昇格 → クーポン・ノベルティを申請
```
