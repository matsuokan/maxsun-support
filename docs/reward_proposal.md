# 継続的特典の提案書

**プロジェクト**: MAXSUN コミュニティサポート  
**作成日**: 2026-03-25  
**ステータス**: 提案（未決定）

---

## 課題

現在の報酬設計はランク到達時の **一回限りの特典**（ノベルティ・クーポン）のみ。ランク達成後に活動を続ける動機が薄い。

```
現状: ランク到達 → 特典1回もらう → 終わり（モチベーション低下）
理想: ランク到達 → 継続的な特典 → 活動を続ける → さらに上のランクへ
```

---

## 提案一覧

### A. コストゼロで始められる特典

| 特典 | 内容 | 対象ランク |
|---|---|---|
| **🔓 先行情報アクセス** | 新製品・βBIOS の情報を一般公開前に共有 | 🥈 サポーター〜 |
| **🗣️ 開発者チャンネル招待** | 製品フィードバックが開発に反映される限定チャンネル | 🥇 エキスパート〜 |
| **📛 専用バッジ＋名前色** | Discord 上で常に目立つ表示（承認欲求を満たす） | 全ランク（設計済み） |
| **🏷️ 称号システム** | 「BIOS マスター」「メモリ博士」等のカスタム称号 | 特定分野で貢献した人 |

### B. 低コストで効果が高い特典

| 特典 | 内容 | 対象ランク |
|---|---|---|
| **🔄 定期クーポン** | ランク維持中は四半期ごとにクーポン配布 | 🥇 エキスパート〜 |
| **📦 新製品レビューサンプル** | 製品を提供 → SNS/YouTube でレビュー投稿 | 💎 アンバサダー |
| **🎁 誕生日特典** | 誕生月に追加クーポン or ノベルティ | 🥈 サポーター〜 |

### C. 中長期で検討する特典

| 特典 | 内容 | 対象ランク |
|---|---|---|
| **💰 アフィリエイトリンク** | 専用リンク経由の売上から継続報酬 | 💎 アンバサダー |
| **🎮 限定イベント** | オフ会・オンラインゲーム大会への招待 | 🥇 エキスパート〜 |
| **🎓 公認サポーター認定** | MAXSUN 公認の「テクニカルサポーター」肩書き | 💎 アンバサダー |

## おすすめ初期構成

**Phase 1（すぐ実施可能）**
- 先行情報アクセス + 開発者チャンネル → コストゼロ
- 称号システム → Discord ロール追加のみ

**Phase 2（β公開後）**
- 定期クーポン（四半期） → 継続動機の本命
- 新製品レビューサンプル → SNS 拡散効果で広告費削減にも

**Phase 3（正式公開後）**
- アフィリエイト・公認サポーター → コミュニティ成熟後に検討
