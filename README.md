# MAXSUN コミュニティサポート

> MAXSUN製品（マザーボード・グラフィックボード）ユーザーのための Discord コミュニティサポートプロジェクト

---

## コンセプト

既存ユーザーが新規ユーザーをサポートし、貢献度に応じて MAXSUN がノベルティ・クーポンで還元するコミュニティ型サポートの仕組みを構築します。

```
新規ユーザー → 質問 → Discord コミュニティ → 既存ユーザーが回答
                                                      ↓
                                           貢献ポイント加算
                                                      ↓
                               ノベルティ・製品割引クーポンで還元
```

---
