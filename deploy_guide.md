# MAXSUN コミュニティBot 導入作業指示書

**対象**: 専用PC担当者  
**作成日**: 2026-04-06  
**所要時間の目安**: 約30〜60分  

> この手順書は、MAXSUNコミュニティのDiscord Botを専用PCに導入・常駐起動させるための手順書です。  
> 不明な点は担当者（zhang）に確認してください。

---

## 事前に受け取るもの

作業を始める前に、担当者から以下の2つのファイルを受け取ってください。

| ファイル名 | 説明 |
|---|---|
| `.env` | BotのトークンやGoogle SheetsのIDが入った設定ファイル |
| `credentials.json` | Google Sheetsと連携するための認証ファイル |

> [!CAUTION]
> これらのファイルは**絶対に他人に見せたり、クラウドにアップロードしたりしないでください。**  
> 流出した場合、セキュリティ上の重大な問題が発生します。

---

## STEP 1: Python 3.11 のインストール

### 1-1. インストーラーのダウンロード

ブラウザで以下のURLを開いて、「Windows installer (64-bit)」をダウンロードしてください。

```
https://www.python.org/downloads/release/python-3119/
```

ページ下部の「Files」セクションから `Windows installer (64-bit)` を探してクリックします。

### 1-2. インストールの実行

1. ダウンロードした `.exe` ファイルをダブルクリックして開く
2. **「Add python.exe to PATH」にチェックを入れる**（← 重要！忘れずに）
3. 「Install Now」をクリックしてインストールを完了させる

### 1-3. インストールの確認

「スタート」メニューから「コマンドプロンプト」を開き、以下を入力してください。

```
python --version
```

`Python 3.11.x` と表示されれば成功です。

---

## STEP 2: Git のインストール

### 2-1. インストーラーのダウンロード

以下のURLから「64-bit Git for Windows Setup」をダウンロードしてください。

```
https://git-scm.com/download/win
```

### 2-2. インストールの実行

インストーラーを起動し、**設定はすべてデフォルトのまま「Next」を押し続けて**インストールしてください。

### 2-3. インストールの確認

コマンドプロンプトを再起動し、以下を入力してください。

```
git --version
```

`git version 2.x.x` と表示されれば成功です。

---

## STEP 3: リポジトリのクローン（ダウンロード）

コマンドプロンプトで以下のコマンドを順番に実行してください。

```
cd C:\
git clone https://github.com/matsuokan/maxsun-support.git
```

完了すると `C:\maxsun-support` というフォルダが作成されます。

---

## STEP 4: 依存パッケージのインストール

引き続きコマンドプロンプトで以下を実行してください。

```
cd C:\maxsun-support
pip install -r bot/requirements.txt
```

いくつかのパッケージがインストールされます。エラーが出なければ成功です。

---

## STEP 5: 設定ファイルの配置

担当者から受け取った2つのファイルを、以下の場所にコピーしてください。

### コピー先フォルダ

```
C:\maxsun-support\bot\
```

### コピーするファイル

| ファイル名 | コピー先 |
|---|---|
| `.env` | `C:\maxsun-support\bot\.env` |
| `credentials.json` | `C:\maxsun-support\bot\credentials.json` |

> [!TIP]
> `.env` ファイルはファイル名の先頭がドット（`.`）で始まるため、エクスプローラーで表示されない場合があります。  
> その場合はコマンドプロンプトで `dir /a C:\maxsun-support\bot\` と入力して存在を確認してください。

### 配置後の確認

コマンドプロンプトで以下を実行して、2つのファイルが表示されれば成功です。

```
dir C:\maxsun-support\bot\.env
dir C:\maxsun-support\bot\credentials.json
```

---

## STEP 6: Botの動作確認（テスト起動）

一度手動で起動して、エラーが出ないか確認します。

```
cd C:\maxsun-support\bot
python main.py
```

起動ログに以下のような表示が出れば成功です。

```
Logged in as MaxBot#xxxx
Synced xx commands
```

> [!NOTE]
> エラーが表示された場合は、画面全体のスクリーンショットを撮って担当者に送ってください。

確認できたら、**Ctrl + C** キーでBotを停止してください。

---

## STEP 7: 電源設定の変更（スリープ無効化）

Botを常時稼働させるため、PCがスリープしないように設定します。

1. スタートメニューを開き「電源とスリープ」と検索して開く
2. 「電源とスリープ」の設定画面を開く
3. **「スリープ」の設定をすべて「なし」に変更する**
   - 「電源に接続しているとき、次の時間が経過後にスリープ状態にする」→ **「なし」**
   - 「電源に接続しているとき、次の時間が経過後に画面をオフにする」→ 任意（「なし」推奨）

---

## STEP 8: タスクスケジューラーへの登録（自動起動設定）

PCの電源を入れると自動的にBotが起動するように設定します。

### 8-1. バッチファイルの確認

`C:\maxsun-support\bot\start_bot.bat` というファイルがすでに存在します。このファイルを使います。

### 8-2. タスクスケジューラーを開く

1. スタートメニューで「タスクスケジューラー」と検索して開く
2. 右側の「タスクの作成」をクリック

### 8-3. タスクの設定

**「全般」タブ**
- 名前: `MAXSUN Bot 自動起動`
- 「ユーザーがログオンしているかどうかにかかわらず実行する」を選択
- 「最上位の特権で実行する」にチェックを入れる

**「トリガー」タブ**
- 「新規」をクリック
- 「タスクの開始」を「スタートアップ時」に設定
- 「遅延時間を指定する」にチェックを入れ、**30秒**に設定
- 「OK」をクリック

**「操作」タブ**
- 「新規」をクリック
- 「プログラム/スクリプト」に以下を入力:
  ```
  C:\maxsun-support\bot\start_bot.bat
  ```
- 「開始（オプション）」に以下を入力:
  ```
  C:\maxsun-support\bot
  ```
- 「OK」をクリック

**「条件」タブ**
- 「コンピューターをAC電源で使用している場合のみタスクを開始する」の**チェックを外す**

**「設定」タブ**
- 「タスクが失敗した場合の再起動間隔」にチェックを入れ、**1分**に設定
- 「再起動の試行回数」を **3回** に設定

8-4. 「OK」をクリックして保存。管理者パスワードを求められた場合は入力してください。

---

## STEP 9: 動作確認

### 再起動テスト

PCを再起動して、数分後にBotがDiscordサーバーに「オンライン」として表示されることを確認してください。

### ログの確認

`C:\maxsun-support\bot\` フォルダ内に `bot.log` ファイルが生成されます。  
内容を確認し、エラーが出ていないことを確認してください。

## トラブルシューティング

| 症状 | 確認すること |
|---|---|
| `python` コマンドが認識されない | STEP 1-2 で「Add to PATH」にチェックしたか確認。Pythonを再インストール |
| `git clone` が失敗する | インターネット接続を確認。Gitが正しくインストールされているか確認 |
| `.env` や `credentials.json` が認識されない | ファイルがSTEP 5の正しい場所にあるか確認 |
| Bot起動時にエラーが出る | エラー画面のスクリーンショットを撮って担当者（zhang）に送る |
| 再起動後にBotが起動しない | タスクスケジューラーを開き、タスクが「準備完了」状態か確認 |
